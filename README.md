# LunarCoreTools
适用于LunarCore的web命令生成器

## 原项目
[GrasscutterTools](https://github.com/wmn1525/grasscutterTools)

## 安装依赖
`
npm install yarn
`

## 开发
`
npm run dev
`

## 编译运行
`
npm run build:pro  
`
`
npm install http-server  
`
`
cd dist  
`
`
http-server  
`

## 直接运行
**可能更新不及时**  
clone gh-pages分支  
`
npm install http-server  
`
`
http-server  
`
